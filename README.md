# My Pages

This repository is the **deployed GitHub Pages build output** for the [`github-pages`](../github-pages) Nuxt 4 source project. It contains the static site generated by that project (e.g. `index.html`, `assets/`, and `particles.json`) and is served directly by GitHub Pages.

The repo includes a `.nojekyll` file so GitHub Pages serves the raw build output without running Jekyll. The generated files here are produced by the source project — they should not be hand-edited; make changes in `github-pages` and regenerate.
